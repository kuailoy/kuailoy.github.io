# shadowsocks 的安装和配置

连接VPS后， 按照以下步骤进行安装和配置

# 安装pip

CentOS 7 自带 python 2，可以通过命令查看 python 版本

[root@vultr ~]# python --version

pip 是 python 的包管理工具， python 版本的 shadowsocks 已发布到 pip 上，我们可以通过 pip 命令来安装shadowsocks。

执行下列命令安装pip

[root@vultr ~]# curl "https://bootstrap.pypa.io/get-pip.py" -o "get-pip.py"
[root@vultr ~]# python get-pip.py

# shadowsocks 安装配置

pip安装完成后，执行下面命令安装shadowsocks

[root@server ~]# pip install --upgrade pip
[root@server ~]# pip install shadowsocks

安装完成后，需要创建shadowsocks的配置文件/etc/shadowsocks.json，编辑内容如下：

[root@server ~]# vi /etc/shadowsocks.json
{
  "server": "0.0.0.0",
  "local_address": "127.0.0.1",
  "local_port": 1080,
  "port_password": {
    "8080": "填写密码",    // 开放端口对应密码
    "8081": "填写密码"     // 开放端口对应密码
  },
  "timeout": 600,
  "method": "aes-256-cfb" // 加密方法，可选aes-128-cfb, aes-192-cfb, aes-256-cfb,
                          // bf-cfb, cast5-cfb, des-cfb, rc4-md5, chacha20, salsa20, rc4, table
}

记录下端口、密码、加密方法，使用shadowsocks客户端时需要配置一致

如果不需要多个端口，可以使用下面的单端口配置

{
  "server": "0.0.0.0",
  "server_port": 8080,    // 开启服务端口
  "password": "填写密码",
  "method": "aes-256-cfb"
}

# 配置shadowsocks自启动

编辑shadowsocks服务的启动脚本，如下

[root@server ~]# vi /etc/systemd/system/shadowsocks.service
[Unit]
Description=Shadowsocks

[Service]
TimeoutStartSec=0
ExecStart=/usr/bin/ssserver -c /etc/shadowsocks.json

[Install]
WantedBy=multi-user.target

执行以下命令启动 shadowsocks 服务：

[root@vultr ~]# systemctl enable shadowsocks
[root@vultr ~]# systemctl start shadowsocks

检查 shadowsocks 服务是否已成功启动，可以执行以下命令查看服务的状态：

[root@vultr ~]# systemctl status shadowsocks -l

正常状态如下

如果修改了配置，需要重启shadowsocks服务

[root@vultr ~]# systemctl stop shadowsocks
[root@vultr ~]# systemctl start shadowsocks

确认服务启动成功后，一定要记得配置防火墙规则开放端口，不然客户端是无法连接的！

[root@vultr ~]# firewall-cmd --zone=public --add-port=8080/tcp --permanent
success
[root@vultr ~]# firewall-cmd --zone=public --add-port=8081/tcp --permanent
success
[root@vultr ~]# firewall-cmd --reload
success

然后安装并运行shadowsocks客户端程序，增加服务器配置就可以上网啦

# 配置BBR加速

BBR是Google 开源的 TCP 拥塞控制算法

部署最新版内核，开启TCP BBR 加速的 VPS，据说网速可以提升几个数量级

实际使用感觉对youtube加速效果挺明显的

# BBR项目地址

https://github.com/google/bbr

# 升级内核

开启 BBR 要求 最新的 Linux 内核版本，可使用如下命令查看当前内核版本：

[root@vultr ~]# uname -r

可得到类似下面的结果

如果当前内核版本低于 4.10，可使用 ELRepo 源更新：

sudo rpm --import https://www.elrepo.org/RPM-GPG-KEY-elrepo.org
sudo rpm -Uvh http://www.elrepo.org/elrepo-release-7.0-2.el7.elrepo.noarch.rpm
sudo yum --enablerepo=elrepo-kernel install kernel-ml -y

安装完成后查看已安装内核

rpm -qa | grep kernel

修改grub2引导

grub2-set-default 1

然后重启

shutdown -r now

查看是否生效

uname -r

可以看到内核版本已经升级到了5.4.10

# 安装Google BBR

echo 'net.core.default_qdisc=fq' | sudo tee -a /etc/sysctl.conf
echo 'net.ipv4.tcp_congestion_control=bbr' | sudo tee -a /etc/sysctl.conf
sysctl -p

结果如下

检查BBR是否成功

sysctl net.ipv4.tcp_available_congestion_control


sysctl -n net.ipv4.tcp_congestion_control


lsmod | grep bbr

看到类似上面的提示，表示BBR已经开启成功了！
