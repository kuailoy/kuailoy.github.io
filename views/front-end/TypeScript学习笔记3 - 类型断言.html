    TypeScript学习笔记3 - 类型断言

    kuailoy 8/25/2019 typescript

    # 类型断言

    有时候，我们需要指定某个值为特殊的类型，以访问该类型的属性，并不需要TypeScript进行类型检查， 例如：

    // 获取联合类型的length
function getLength(target: string | number): number {
  if (target.length) {
    // error 类型"string | number"上不存在属性length
    // 类型number上不存在属性length
    return target.length; // error 同上
  } else {
    return target.toString().length
  }
}

    这时我们可以使用类型断言， 语法：

    <类型>值

    或者

    值 as 类型

    Tslint推荐使用as语法，并且在JSX中只能使用这种写法

    使用类型断言解决上面的报错

    function getLength(target: string | number): number {
  if ((target as string).length) {    // as语法
    return (<string>target).length;   // 尖括号语法
  } else {
    return target.toString().length
  }
}
